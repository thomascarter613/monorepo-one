# Documentation Index

This directory contains the durable planning, architecture, governance, implementation, and handoff documentation for the monorepo.

## Primary Sections

```text
docs/
├── project/      # Product, planning, principles, requirements, and delivery docs
├── adr/          # Architecture Decision Records
└── ai/           # AI-readable current state, handoff, next steps, and open questions
```

## Documentation Standard

Documentation should be:

* accurate
* versioned
* maintained with the code
* useful to humans
* useful to AI assistants
* explicit about assumptions and unknowns
* tied to verification where applicable
