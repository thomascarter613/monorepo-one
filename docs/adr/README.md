# Architecture Decision Records

Architecture Decision Records capture important technical and organizational decisions.

## ADR Rules

Each ADR should include:

* status
* context
* decision
* consequences
* alternatives considered
* verification or acceptance criteria where applicable

## Status Values

Recommended ADR statuses:

* proposed
* accepted
* superseded
* deprecated
* rejected
